# Fast API Quick Start Template

|   by [Mahimai Raja J](www.github.com/mahimairaja)

## Installation

```bash

$ poetry intsall
```

## Usage 

1. Modify the variables in the `.env` file and `main.py`. And run the following command to start the server.

```bash
uvicorn main:app --reload
```
